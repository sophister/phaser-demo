# phaser-demo

javascript 2d game demo based on Phaser game framework


需要在全局安装 `npm install http-server -g` , 然后在项目根目录执行 `npm start` 来启动本地server

本demo基于  `Phaser 2.6.2` 版本, 即 2.X 中最后一个官方正式版!
